# Personal Website

A modern personal website built with React and Vite, inspired by [Lucy Ling's website](https://github.com/LucyLing24/longling).

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The built files will be in the `dist` folder.

## Deploy to GitHub Pages

```bash
npm run deploy
```

## Features

- Modern React + Vite setup
- Responsive design
- Smooth scrolling navigation
- Clean, minimalist UI

## Customization

Edit the components in `src/components/` to add your personal information:

- `Hero.jsx` - Your name, title, and contact links
- `About.jsx` - About me section
- `News.jsx` - News and updates
- `Publications.jsx` - Your publications
- `Internship.jsx` - Work experience
- `Education.jsx` - Education background
- `SelectedAwards.jsx` - Awards and achievements
